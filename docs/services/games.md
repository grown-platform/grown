# Games

A built-in arcade of **109 browser games** served as static, self-contained HTML under `web/app/public/games/`. No backend or account is required — every game runs fully offline and the collection is installable as its own PWA. This gallery was captured live with Playwright (Chromium) against a local static server on port 8099; each game was screenshot at desktop (1440x900) after a ~1.5s settle for canvas games to draw.

Each thumbnail links to its full-size capture. Mobile (390x844) captures exist alongside every desktop one in [`screenshots/games/`](screenshots/games/).
