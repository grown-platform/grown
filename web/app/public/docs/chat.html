Communication

Chat

Real-time team chat — channels and direct messages that update live, without handing your conversations to a third party.

Open Chat → Browse all apps

Overview

Chat gives your workspace a fast place to talk. Group channels keep teams in sync; direct messages cover quick one-on-ones. Messages stream over a live connection, so replies, reactions and presence appear the moment they happen. The same chat is also embedded as a compact rail inside Mail.

Features

  • Channels & direct messages. Create named group channels with multiple members, or one-on-one DMs — including a private note-to-self channel.
  • Real-time delivery. Each channel holds a live websocket connection, so new messages, deletions and presence arrive instantly.
  • Presence. See how many people are currently active in a channel.
  • Threads. Reply under any message to keep side conversations tidy; the parent shows its reply count.
  • Reactions. React with a row of quick emoji or pick any emoji; reactions show a count and whether you reacted.
  • Attachments. Drag-and-drop or paste files and images; images render inline, other files become download links.
  • Rich formatting. Write in Markdown — bold, italic, strikethrough, code blocks, lists and tables — with a formatting toolbar and emoji picker in the composer.
  • @mentions. Mention teammates with autocomplete that searches the org directory by name or email.
  • GIF search. Find and send GIFs without leaving the composer.
  • Unread badges. Per-channel unread counts that clear when you open the channel.
  • Embedded rail. A compact chat sidebar that drops into other apps such as Mail for quick messages without switching context.

How it works

Chat keeps a websocket open per channel for live message, presence and deletion events. Attachments upload to your workspace's object storage and are referenced inline. The member directory and @mention autocomplete are served from the shared org directory, and every request rides your workspace single sign-on.

Transport
Live websocket per channel (messages, presence, deletions)
Storage
Messages in the workspace database; attachments in object storage
Directory
Shared org directory powers member search and @mentions
Auth
Workspace single sign-on (OIDC)
