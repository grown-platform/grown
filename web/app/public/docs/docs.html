Docs

Collaborative rich-text documents, edited in real time — write together, comment, track versions and export anywhere.

Open Docs → Browse all apps

Overview

Docs is a full collaborative word processor. Several people can edit the same document at once — you see each other's live cursors and selections — backed by a conflict-free sync engine so nobody's work is lost. Start from a blank page or pick a starter template, then format, comment and share without leaving the workspace.

Features

  • Real-time co-editing. Multiple people edit the same document together, with live cursors, selections and presence avatars in the header.
  • Comments. Anchor comments to a selection, reply in threads, and resolve or reopen them — all in a side panel.
  • Version history. Automatic snapshots with author and timestamp; name a version, preview it, and restore any earlier state in one click.
  • Sharing & permissions. Share by link as viewer or editor, or invite specific people by email with per-user can-edit / can-view access — works across organisations.
  • Starter templates. Begin from Blank document, Project proposal, Meeting notes, Brochure, Newsletter or Business letter.
  • Rich formatting. Headings 1–6, bold/italic/underline/strikethrough, super/subscript, text and highlight colours, fonts and sizes, line spacing, alignment, indentation and clear-formatting.
  • Lists, tables & media. Bulleted, numbered and check lists; insert tables, images (by URL, paste or drag-and-drop), links and horizontal rules.
  • Export anywhere. Download as Word (.docx), OpenDocument (.odt), RTF, PDF, plain text, HTML, EPUB or Markdown.
  • Find & replace, command palette. Search across the document and search every menu action from one box (Alt+/), with a full keyboard-shortcut reference (Ctrl+/).
  • Page layout. Ruler with margin and indent controls, portrait/landscape orientation, optional page numbers, and an Editing/Viewing mode toggle.

How it works

Docs builds on a rich-text editor synced through a conflict-free replicated data type (CRDT). Every keystroke merges cleanly across all editors over a live websocket, so concurrent edits never collide. Document content, comments and version snapshots persist in the workspace; the same single sign-on grants access across the platform, including sharing with people in other organisations.
