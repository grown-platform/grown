Account

Settings

Your personal control panel — manage your profile and photo, mint scoped API tokens, and tune how the workspace looks and behaves for you.

Open Settings → Browse all apps

Overview

Settings is where you manage your account — distinct from the org-wide Admin console. Edit your name, username, phone and email, upload an avatar, create personal API tokens for scripts and integrations, and set language, formats, appearance and notification preferences that follow you across the workspace.

Features

  • Profile. Edit your first and last name, a unique username, phone number and email. Changing your email sends a verification link, and your address shows a verified / unverified badge.
  • Profile photo. Upload an avatar; it updates everywhere your identity appears.
  • API tokens. Create personal access tokens that authenticate scripts and integrations as you over the HTTP API (Authorization: Bearer <token>). The full token is shown once — copy it then.
  • Token scopes. Grant full access, or limit a token to chosen services (Drive, Mail, Calendar, Contacts, Sheets, Docs, Photos, Music, Video, Tasks), optionally read-only. Set an expiry — never, 30 days, 90 days or a year — and revoke any token at any time.
  • Language & region. Pick your language, date and time formats, and which day the week starts on.
  • Appearance. Choose comfortable or compact density, and the default app you land on after signing in.
  • Notifications. Toggle email digests and activity summaries.

How it works

Settings reads and writes your account over per-user endpoints (/api/v1/me/*). Profile and email changes flow through Zitadel — email changes stay pending until you verify the new address. Preferences are saved server-side and applied app-wide (density, for instance, is set on the page so components react immediately). API tokens are hashed; the raw value is returned only once at creation.

Scope
Per-user — your account only
Identity
Zitadel (name, username, phone, verified email)
Tokens
Scoped bearer tokens, shown once, revocable
