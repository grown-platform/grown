Photos

A photo library with albums, a full-screen viewer and a built-in editor — backed by your own object storage.

Open Photos → Browse all apps

Overview

Photos is your workspace's gallery. Upload images, browse them in a grid, gather them into albums, and open any one full screen. A built-in editor handles quick crops and adjustments, and rich metadata shows the camera and capture details — all stored in object storage you run, with Immich integration.

Features

  • Upload. Add photos by drag-and-drop or file picker.
  • Albums. Create albums, add and remove photos, set a cover image, and delete albums while keeping the photos.
  • Views. Browse all photos, just favorites, or a single album in a responsive grid.
  • Lightbox. A full-screen viewer with prev/next, keyboard arrows and swipe, plus star, add-to-album, info, edit, download and delete actions.
  • Built-in editor. Rotate, adjust brightness and contrast, and crop — then save the result as a new copy.
  • Photo metadata. See dimensions, camera make and model, lens, capture date and exposure settings.
  • Favorites. Star photos and find them in the Favorites view.
  • Batch actions. Multi-select to add to an album, remove from an album, or delete.
  • Editable details. Give a photo a description and view its name, size, type and creation date.
  • Download. Save any photo straight from the viewer, menu or info dialog.

How it works

Photos and albums are served from the workspace API, with image content stored in object storage and integrating with Immich. Uploads go up as multipart form data; favorites and description edits apply optimistically and sync to the server. The editor renders crops and adjustments in the browser and saves the output as a new photo. Everything is gated by your workspace single sign-on.
