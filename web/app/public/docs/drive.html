Files & media

Drive

File storage and sharing with previews, versions and rich photo metadata — plus share links you can set to expire.

Open Drive → Browse all apps

Overview

Drive is your workspace's file storage. Organise files in folders, preview them in the browser, keep a full version history, and share them — with specific people or via a link that can expire. Image files surface their EXIF metadata, including camera, date taken and GPS location.

Features

  • Upload & organise. Drag-and-drop or pick files, create folders, rename, move, copy and trash — with grid and list views.
  • In-browser previews. View PDFs, images, video, audio, CSV tables and text/JSON without downloading.
  • EXIF photo metadata. Image previews show dimensions, camera make and model, lens, focal length, ISO, exposure, date taken and GPS location with a map link.
  • Version history. Every file keeps its prior versions; view, download or restore any version, and restoring saves the current content as a new version.
  • Sharing. Grant access to specific Grown users, or create an "anyone with the link" share with a viewer, commenter or editor role.
  • Expiring share links. Set a link to never expire or to lapse after 1 hour, 1 day, 7 days or 30 days; revoke shares any time.
  • File details panel. See who has access, file type, size, owner, location and timestamps in a side panel.
  • Smart views. My Drive, Shared with me, Starred, Recent and Trash, plus search and filters by name, type and owner.
  • Starred files. Flag favorites and find them in the Starred view.
  • Open in editors. Hand files off to the matching Grown editor (for example, a spreadsheet opens in Sheets).

How it works

Drive stores file content and metadata in your workspace, with each file tracking its versions. Shares are token links carrying a role and an optional expiry, reachable at /drive/share/<token>, alongside per-user access grants. Previews and EXIF parsing run in the browser, and everything is gated by your workspace single sign-on.
