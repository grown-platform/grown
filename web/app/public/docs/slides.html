Slides

Build and present slide decks — design slides with text, shapes, images and animations, then present full-screen with speaker notes.

Open Slides → Browse all apps

Overview

Slides is a presentation editor and deck library in one. Design each slide on a free-form canvas — drag and resize text boxes, shapes and images — add transitions and per-element animations, write speaker notes, then run the deck in a full presenter view. Decks are collaborative, so a team can build the same presentation together in real time.

Features

  • Free-form canvas. Drag, resize and arrange text boxes, images, lines and shapes (rectangle, rounded rectangle, ellipse, triangle, diamond, arrow).
  • Text styling. Choose font, size and colour; bold/italic/underline; horizontal and vertical alignment — edit text in place with a double-click.
  • Templates. Start from Blank, Simple Light, Bold Dark, Photo Focus or Pitch Deck.
  • Transitions & animations. Slide transitions (fade, slide, push) and per-element entrance animations — appear, fade in, fly in from bottom or left — with click ordering.
  • Speaker notes. Per-slide notes shown in the editor and in the presenter view.
  • Presenter view. Full-screen presenting with the current slide, next-slide preview, notes, an elapsed timer and a slide/step counter; step through animations with arrow keys or space.
  • Arrange & edit. Cut/copy/paste and duplicate elements, bring to front / send to back, nudge with arrow keys, and undo/redo.
  • Deck library. My presentations and Shared-with-me views; create, rename, duplicate or trash decks, with live thumbnails in the slide panel.
  • Export. Download as PowerPoint (.pptx), PDF, plain text (with notes), HTML, or the current slide as PNG, JPG or SVG.
  • Real-time collaboration. Co-edit decks with live presence indicators showing who's on which slide.

How it works

Each deck is a list of slides, and each slide is a set of positioned elements — text, shapes and images — with its own background, notes, transition and animation order. The editor renders that model on a scalable canvas and replays it in the presenter view. Edits broadcast to other collaborators over a live websocket, and sharing is governed by the same workspace sign-on, including people in other organisations.
