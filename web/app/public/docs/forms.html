Forms

Build forms and surveys, collect responses, and see the results — with an optional quiz mode that grades itself.

Open Forms → Browse all apps

Overview

Forms lets you build a form or survey from a stack of question types, share a link, and collect responses — then read them one by one or as a summary with charts. Add sections for multi-page forms, branch between them based on answers, or turn the whole thing into a self-grading quiz.

Features

  • Nine question types. Short answer, paragraph, multiple choice, checkboxes, dropdown, linear scale, date, time and file upload.
  • Sections & branching. Split long forms into named sections (pages) and branch to a different section — or submit early — based on a multiple-choice or dropdown answer.
  • Question options. Mark questions required, add descriptions, configure linear-scale endpoints and labels, and reorder or duplicate questions.
  • Form settings. Collect respondent email, limit to one response, show a progress bar, shuffle question order, toggle accepting responses, and set a custom confirmation message.
  • Filling experience. Multi-page navigation with Back/Next, a progress bar, retained answers and required-field validation.
  • Responses. Browse individual responses with Previous/Next, or open a Summary tab with bar charts for choice questions and lists of text answers.
  • Quiz mode. Set correct answers and point values, auto-grade on submission, show scores to respondents, and see average / high / low across all responses.
  • Export. Download every response as a CSV, including email, answers and quiz scores.
  • Templates & management. Start from a template (Contact info, Party invite, RSVP, Event registration, Course evaluation, Event feedback), or duplicate, preview and share any form.

How it works

A form is an ordered list of questions grouped into sections; each submission stores its answers as structured data, with branching resolved as the respondent moves between pages. Choice answers roll up into summary charts, and in quiz mode answers are scored against the answer key on submission. Sharing a form gives respondents a link, while building and viewing results use the workspace single sign-on.
