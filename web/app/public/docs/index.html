Overview

Workspace bundles the everyday tools a team needs into a single self-hosted suite. Sign in once and everything — files, mail, calendar, real-time docs, video meetings, even a phone system — is a click away from the launcher.

What's new recent

A selection of recently shipped improvements.

  • Installable + offline PWA. The whole suite installs to your phone's home screen and launches offline (cached app shell).
  • Meet in Calendar. Add a video meeting to a calendar event (new or existing). When the first person joins, invited attendees get a "Meeting started" alert.
  • Drive photo metadata. Image previews show dimensions, camera, lens, date taken, exposure and GPS location (with a map link) read from EXIF.
  • Sheets, supercharged. A 245-function engine with dynamic-array spill, data validation, color-scale & data-bar conditional formatting, charts and pivot tables.
  • Telephony. A softphone and texting UI, with a full 3CX-style PBX administration console gated to admins. Working call recordings and a live activity log.
  • Native game ports. Mighty Mike (a remake of Pangea's Power Pete) compiled to WebAssembly with a floating touch joystick and a D-Pad/Joystick switch.
  • Security & ops. Demo-account audit logging with IP + geolocation, configurable API rate limiting, and proper asset caching for fast, offline-friendly loads.

Install & offline

Workspace is a Progressive Web App, so it installs like a native app and keeps working without a connection.

  • Install on a phone. Open workspace.pick.haus in your browser → menu → Add to Home Screen. It installs as "Workspace" with its own icon and opens full-screen.
  • Offline. Once installed, the app launches and renders even with no connection — the interface and its assets are cached on the device. (Live data such as new mail still needs a connection.)
  • Games offline. The Games hub and game ports cache their files, so they play with no signal.

For administrators

Run the platform with visibility and control.

  • Users & services. Add users and enable or disable individual apps per org.
  • Audit logs. Every change is recorded with the actor's email, IP address and resolved geolocation — searchable by email and service.
  • Rate limiting. Sensible API rate limits with stricter limits on auth endpoints, all configurable.
  • Telephony admin. A complete PBX console (extensions, ring groups, queues, routes, recordings, reports) — visible only to admins.
