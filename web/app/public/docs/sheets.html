Sheets

Powerful collaborative spreadsheets — 245+ functions, dynamic-array spill, data validation, conditional formatting, charts and pivot tables.

Open Sheets → Browse all apps

Overview

Sheets is a full-featured spreadsheet you and your team can edit together in real time. Behind the grid is a deep formula engine plus the analysis tools you expect — validation, conditional formatting, charts and pivot tables — so you can go from a list of numbers to a finished report without leaving the workspace. Start from blank or a ready-made template.

Features

  • A deep formula engine. 245+ functions with dynamic-array spill, so a single formula can fill a whole range of results.
  • Data validation. Restrict cells with dropdown lists, checkboxes, number, text-length, text-contains and date rules — with an option to reject invalid input.
  • Conditional formatting. Single-colour conditions (greater/less/equal/contains/between), 2- and 3-colour scales, and in-cell data bars.
  • Charts. Insert column, bar, line, area and pie charts with titles, multi-series data and a live preview, embedded in the sheet.
  • Pivot tables. Summarise data by row and column fields with Sum, Count, Average, Min or Max aggregation, plus automatic row, column and grand totals.
  • Named ranges. Give A1 ranges memorable names and reuse them in formulas.
  • Find & replace. Match case, whole-cell or regular expressions, scoped to the sheet, the whole workbook or a selection.
  • Sort, filter & clean. Sort sheets or selections A→Z / Z→A, add filters, split text to columns, and randomise ranges.
  • Freeze panes. Freeze rows and/or columns up to the current cell so headers stay in view.
  • Formatting. Number, currency, percent, scientific, date and time formats; bold/italic/underline; alignment; and merge cells.
  • Templates. Start from Monthly budget, To-do list, Weekly schedule or Expense tracker.
  • Export. Download as Excel (.xlsx), OpenDocument (.ods), PDF, HTML, CSV or TSV.
  • Real-time collaboration. Edit together with live presence and per-user cursors.

How it works

Sheets pairs a spreadsheet grid with a function engine and a set of analysis dialogs — validation, conditional formatting, charts and pivots — whose definitions are stored alongside the workbook so they survive save and reload. Edits sync between collaborators over a live websocket, and the same workspace sign-on controls who can open and edit each file.
