Communication

Groups

Mailing lists and team groups — a shared address and a place for the whole group to talk.

Open Groups → Browse all apps

Overview

Groups bundle people behind a shared name and email address. Each group has a member roster and a stream of topic-based conversations, so a team, project or community has one place to reach everyone and keep discussions together.

Features

  • Groups with an address. Create a group with a name, email address and optional description.
  • Member rosters. Add members from the org directory with live search by name or email; manage the roster at any time.
  • Topic conversations. Start topics with a subject and an opening message; each group lists its topics with author, post count and last activity.
  • Posts. Reply to a topic to keep the thread going, with author and a "time ago" stamp on every post.
  • Search. Find groups by name, email or description.
  • Tabbed group view. Conversations, About (description, email, created date) and Members tabs for each group.

How it works

Groups are stored per organisation and served from the workspace API. Member search resolves against the shared org directory, so any user in your org can be found and added — even before they've signed in. Each group owns its topics, and each topic its sequence of posts.
